#  Week-1 Project – To-Do List App

About the Project

The To-Do List Web Application helps users manage and organize their daily tasks efficiently. 
Users can add, delete, and mark tasks as completed, and the tasks are stored in the browser's localStorage so they remain available even after refreshing the page.

The goal of this project was to practice core concepts of HTML, CSS, JavaScript, and DOM manipulation while also learning project structure and GitHub workflow.


Features

-  Add new tasks
-  Mark tasks as completed
-  Delete tasks
-  LocalStorage support (tasks remain after refresh)
- Responsive UI

How the Project Works

1. The user types a task in the input field.
2. Clicking the Add button adds the task to the list.
3. Each task shows two buttons: Complete and Delete.
4. The Complete button marks the task as done.
5. The Delete button removes the task from the list.
6. All tasks are saved in localStorage so they stay even after refreshing the page.
